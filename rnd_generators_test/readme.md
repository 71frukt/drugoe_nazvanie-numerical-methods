## О тестировании ГСЧ
Фактически этот проект является библиотекой для тестирования произвольных генераторов случайных чисел. В качестве примера будет протестирован написанный в прошлом задании генератор $\chi^2$.

## Принцип тестирования
Тестирование любого ГСЧ сводится к тестированию разными методами некоторой последовательности, которую выдает генератор. Для возможности проверки ГСЧ разными тестами, используя один инструментарий, используется метод двухуровневого тестирования:
* Имеется последовательность из $N$ чисел на отрезке $[0, 1]$ - $\{x_j\}$. Она разбивается на последовательность из $k$ блоков - $\{block_i\}$.
* Строится тест, как-то сопоставляющий входному блоку $\{block_i\}$ число $Y_i$ - тестовую статистику.
* Этот тест должен знать, какую функцию распределения $F(Y)$ имеет статистика для гипотетического идеального генератора.
* Тогда $\{F(Y_i)\}$ должна иметь равномерное распределение, что алгоритму и остаётся проверить тестом Колмогорова-Смирнова или Андерсона-Дарлинга.

Метод удобен тем, что является инструментарием для любого теста, поскольку сводит тестирование к проверке распределения на равномерность.



Не вдаваясь в подробности, отметим, что возвращаемое значение метода запуска теста есть смысл лишь оценивать, а точнее смотреть, насколько оно близко к 0 или к 1. То есть любые значения результата, лежащие в $[ 0 + \alpha; 1 - \alpha]$ можно считать индикатором качества генератора. Как правило $\alpha$ выбирается порядка $0.02$.

## Пример плохого генератора, проходящего тест KS
Этот генератор обладает идеальными статистическими показателями распределения, но полностью лишен случайности:
$$x_n = \frac{n \pmod M}{M}$$
где $M$ — количество шагов (размер периода). При больших значениях $M$ (например, $10^4$) последовательность имитирует непрерывное равномерное распределение $U(0, 1)$.

Генератор так хорошо проходит тест KS по нескольким причинам:
* Инвариантность к перестановкам: критерий KS вначале своей работы сортирует элементы, то есть работает с ней как с множеством элементов.
* Функиця распределения представляет собой идеальную ступенчатую аппроксимацию прямой $F(x) = x$. Максимальное отклонение $D_n$ составляет $\approx \frac{1}{M}$, что при $M=1000$ дает $D_n = 0.001$, гарантируя $p\text{-value} \approx 1.0$.
* Математическое ожидание последовательности на интервале $[0, 1]$ с шагом $1/M$ в точности совпадает с теоретическим значением $0.5$ (с погрешностью $O(1/M)$).

на выходе мы получим:
```zsh
KS Statistic (Dn): 0.001
P-Value:           1
```

То есть генератор идеален. Очевидно, что этот генератор не пройдёт второй уровень тестирования - мы получим $p-value_2 = 0$, так как выборка {$p-value$} не будет равномерно распределенной на [0, 1]. Поймём, что идеальное распределение было "идеально" в каждом тесте, то есть генератор слишком идеален, чтобы быть случайным.
