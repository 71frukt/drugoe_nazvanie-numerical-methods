# Задача 4 - исследование катастрофиечкой потери точности

Было необходимо исследовать проблему катастрофиечкой потери точности, реализовав три метода для вычисления оценки дисперсии в `float` и `double` с использованием n выборок из чисел из нормального распределения с средним `mid` = `i`, среднеквадратическим отклонением `deviation` = $1/\texttt{i}$.

## Сборка проекта

```bash
cmake -B build
cmake --build build
```

## Запуск
```bash
./build/task4_FloatingPoint
```

|       OPTIONS                    |                                            |
|----------------------------------|--------------------------------------------|
| `-h`, `--help`                   |   Вывести помощь                           |
| `-o`, `--output_file` `<FILE>`   | Задать имя выходного файла                 |
| `-n`, `--gen_num` `<number>`     | Задать количество выборок для каждого типа |
